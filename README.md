How to Run the Website
Since these are pure HTML files, you do not need any complex servers, databases, or build tools to run them. Your web browser is all you need.

Step 1: Set up your project folder
Create a new folder on your computer and name it something like my-portfolio.

Step 2: Create your files
Inside that folder, create two plain text files:

Name the first one index.html.

Name the second one contact.html.
(Note: Ensure your computer is showing file extensions so you aren't accidentally creating index.html.txt.)

Step 3: Add the code

Open index.html in any text editor (like VS Code, Notepad, or TextEdit) and paste the code for the Home Page from the previous response. Save the file.

Open contact.html in your text editor, paste the code for the Contact Page, and save.

Step 4: View it in your browser
Simply double-click index.html, or drag and drop the file directly into an open window of Chrome, Firefox, Safari, or Edge. You can click the "Contact" link in the navigation to seamlessly click over to your contact.html page.
